About BeaverSploit

BeaverSploit is an open-source cybersecurity framework designed for modularity and autonomy. It provides advanced tools for network analysis, penetration testing, and automation. You must have Ruby installed

Available Modules

  • Ping - Ping target for testing servers or websites
  • Packet Injector - Injects custom packets into network traffic
  • Sql-Injection - Injects SQL for testing websites
  • Scanners - Scanners for scanning ports, hosts and dns
  • Xss-scan - XSS scan for testing vulnerabilities
  • Sql-scan - Sql scan for testing vulnerabilities
  • Directory-bruteforce - Directory bruteforce for scanning hidden directories
  • More - Every module on https://github.com/SashaVoden/BeaverSploit-Ruby/
